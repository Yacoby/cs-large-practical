This builds both a file logger and a stderr logger and adds them to the static Logger class that this function creates. This allows use of the created Logs through the functions in the Logger class

void printAllocatedClasses ( )

Prints a list of allocated classes and their current allocation count to stdout.

Used with the command line options to allow an overview of what hasn't been released although it turns out to be a bit useless in some cases due to the gnustep runtime purposefully leaking to avoid mutex locking.

This function doesn't make the assumption that a pool exists
