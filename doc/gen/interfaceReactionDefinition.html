Detailed Description

Immutable description of a reaction containing the Kinetic Constant and Formula.

The class has methods to allow calculations based on this reaction if the apropraite simulaton state is provided.

- (void) applyReactionToCounts: (NSMutableDictionary*)  counts

Applies the current reaction to the molecule counts, altering them.

Parameters
countsthe current molecule counts of the simulation which will be altered to reflect this reaction happening

The alteration to the state happens in place rather than doing the more elegant thing of returning a new state to avoid the overhead of memory allocation during the simulation step. If needed it is trivial to add a wrapper that returns a new state without altering the current state.
