Detailed Description

hold a reaction and its assoicated rates for use in an NSArray

This is used to avoid having to maintian more than one NSMutable arrays for the reaction and the rate which vaslty increasses the possibility for bugs.

The partial sum is used for holding the sum of rates up to the point where this reaction has been considered. As such it is only really useful when in an array.
