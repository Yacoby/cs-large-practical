Detailed Description

Holds a copy of the configuration, which may or may not be valid.

This provides methods to validate the configuration. If the configuration is not valid then it shouldn't be used (i.e. call validate).

The ReactionDefinition objects provided by this class are generated on demand so for the same reaction there may be two different objects.
