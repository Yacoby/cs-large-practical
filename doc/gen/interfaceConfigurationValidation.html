Detailed Description

Holds the result of a validation of SimulationConfiguration.

This is used as the result of the validate (SimulationConfiguration): and allows a distinction between critical errors (which means that the Configuration is invalid) and warnings which means that it has some issues that will not cause a problem with using the configuration.
